GET /api/search

A web search across multiple services.

param name type description possible values
q * string Search query
services * string, multivalued (csv) A services to search in bing, flickr, giphy, ipernity, slideshare, vimeo, youtube
media_types * string, multivalued (csv) A content type to search for text, video, image, audio, presentation
search_in string text, tags
date_from string format: yyyy-MM-dd HH:mm:ss
date_till string format: yyyy-MM-dd HH:mm:ss
ranking string date, relevance, interestingness
page number 1..100
per_page number 1..500
language string A two letter language code en, de, es, fr, ...
extras string, multivalued (csv) statistics, duration, tags
timeout number
